¡Plataforma educativa de e-learning para continuar tu formación en línea!

1. ALUMNADO (Usuarios/as matriculados en ITACA)

El nombre de usuario es vuestro NIA. La contraseña inicial son las 3 primeras letras del apellido en minúscula, seguida de vuestra fecha de nacimiento (ddmma). Al acceder por primera vez, tenéis que cambiar vuestra contraseña.

Si tenéis problemas con la contraseña, clicad el enlace «¿Has olvidado el nombre de usuario o la contraseña?» que encontraréis en la página de acceso de vuestro sabor de Aules y seguid las instrucciones.

En caso de no recibir un correo de confirmación, contactad con el profesorado del curso.

En caso de recibir un correo indicando que «en este lugar no se pueden cambiar las contraseñas», probad con el método 2.

2. DOCENTES (usuarios/usuarias GVA)

Acceded a Aules con vuestro DNI/NIE y la clave de ITACA.

Si no recordáis vuestra contraseña, seguid este enlace: https://acces.edu.gva.es/escriptori

Desde este enlace podréis reiniciar vuestra contraseña mediante uno de estos métodos: DNI/NIE, Identitat Digital, certificado digital o Cl@ve.

3. Otros usuarios/usuarias

Para usuarios con un acceso diferente, si no recordáis vuestro nombre de usuario o la contraseña, poned una incidencia a gvaSAI.
